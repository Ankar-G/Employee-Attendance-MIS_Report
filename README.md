# Employee Attendance & Absenteeism MIS Report

## Overview

This project demonstrates the development of an Excel-based Management Information System (MIS) report for monitoring employee attendance, absenteeism, leave utilization, and workforce performance. The report consolidates attendance records into structured management reports, enabling HR and operations teams to analyze attendance trends and make informed workforce planning decisions.

## Business Objective

Organizations often maintain attendance records across multiple files, making reporting time-consuming and prone to errors. This MIS report streamlines data consolidation and presents key workforce metrics in a structured format to support operational reporting and decision-making.
